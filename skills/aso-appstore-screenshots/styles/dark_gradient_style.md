# Dark Gradient Style System

Premium dark-mode mobile app marketing screenshot style.

## Overall Style
- Futuristic premium health-tech aesthetic
- Dark glassmorphism UI with neon cyan/teal highlights
- High-contrast black and deep charcoal background
- Large bold white typography with selective cyan/teal emphasis
- Rounded app-store screenshot layout, optimized for vertical phone mockups
- Clean, modern, polished, conversion-focused design

## Color Palette
- Background: near-black `#090D0F`, charcoal `#101719`, dark teal `#062B2B`
- Primary accent: bright cyan / aqua `#00CDEB`
- Secondary accent: mint/green `#00F0A8`
- Supporting colors: deep blue, teal gradients, muted purple, amber, red for health metrics
- Text: pure white for headlines, light gray for subtitles, muted gray for secondary UI labels
- Use subtle cyan glow around key elements

## Background Treatment
- Use a dark gradient background blending black, teal, and cyan
- Add large abstract curved ribbon shapes / loop strokes in dark teal with low opacity
- Add soft radial glows behind the phone mockup
- Background should feel dimensional but not busy
- Use vignette darkening at the edges

## Phone Mockup
- Center or slightly offset a realistic iPhone mockup with Dynamic Island
- Use tall portrait framing
- Phone screen should show the app UI in dark mode
- Add subtle shadow and glow behind the device
- Allow parts of the phone to crop off the bottom for dramatic depth
- Phone can overlap with headline area where useful

## Typography
- Use very large bold sans-serif headlines (SF Pro Black / Inter Black / Google Sans Black)
- Headlines should be stacked across 2–3 lines
- Use white text with occasional cyan underline, glow, or highlighted word
- Subtitle should be smaller, clean, centered, and white/light gray
- Use short punchy feature claims
- Use decorative black rounded label stickers behind one keyword when helpful

### Headline Style Examples
- "Weight Loss Forecast"
- "Pen Inventory Management"
- "Dosage Effectiveness Analysis"
- "Log Side Effects"
- "Analyze Your Pain Levels"
- "Personalized GLP-1 Macros"
- "Daily Macro Tracking"
- "Live Medication Levels"

## Text Layout
- Keep text large and readable at Play Store/App Store preview size
- Main headline usually top or bottom third
- Subtitle directly below headline
- Phone mockup occupies the middle and overlaps with headline area where useful
- Use enough spacing; avoid clutter

## UI Card Style
- Dark translucent cards with glass blur effect
- Rounded corners, usually 20–32px
- Thin borders using cyan, teal, gray, or feature-specific colors
- Soft inner glow and outer shadow
- Use gradient panels for highlighted stats
- Key metric cards can float outside the phone screen as callouts
- Floating cards should have bold numeric values and simple icons

## Visual Motifs
- Cyan/teal gradient buttons
- Neon glow highlights
- Abstract medical/health icons
- Line charts with bright cyan/yellow/green trends
- Macro cards with colored accents:
  - Protein: blue
  - Carbs: amber/orange
  - Fat: purple
  - Calories: red/orange
  - Water: cyan
- Use floating overlay stat cards to showcase metrics

## Composition (Layered Depth)
1. Dark abstract background with gradient
2. Blurred enlarged UI / device shadow
3. Main phone mockup with app screenshot
4. Floating cards / callouts (key metrics pop out)
5. Headline and subtitle text

- Allow overlap between phone, cards, and text
- Maintain strong visual hierarchy: headline first, phone second, feature detail third
- Screens should feel consistent as a set

## Lighting
- Add soft cyan/mint glow from corners or behind the phone
- Use subtle highlights on card borders
- Keep the overall mood dark, premium, and energetic

## Rules
### Do:
- Make it look like high-end Play Store promotional screenshots
- Prioritize bold feature communication
- Use sharp, modern, polished spacing
- Make every screen feel part of the same brand system
- Use realistic app screenshots inside the phone

### Avoid:
- Bright white backgrounds
- Flat material design without glow/depth
- Too many colors competing with cyan/teal
- Tiny unreadable text
- Generic wellness-app pastel styling
- Overcrowded compositions
